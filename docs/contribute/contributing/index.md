# Contributing

Thanks for your interest in contributing! This is a community-driven project and
we welcome all contributions.

## Quick Start

**Ready to contribute? Here's how:**

1. **Browse open issues** or start a discussion on the project's GitHub repository
2. **Fork the repository** and make your changes
3. **Submit a pull request** with a clear description

That's it! We'll review and work with you to get it merged.

## Pull Request Guidelines

**Keep it simple:**
- **One change per PR** — easier to review and merge
- **Reference an issue** — link to related discussions
- **Write clear commits** — use format: `Issue #123 - Brief description`
- **Test your changes** — ensure everything works

**Review process:**
We'll check for clarity, correctness, and compatibility. Address feedback
promptly and we'll get your contribution merged!

**First time?** Check out [this guide](https://kcd.im/pull-request) on creating
pull requests.
