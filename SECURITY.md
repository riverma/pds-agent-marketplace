# 👮‍♀️ Security Policy

This document describes the security policy of the PDS Agent Marketplace.

## 📀 Supported Versions

Security updates are applied to the latest released version of this project.

| Version | Supported |
|:--------|:---------:|
| 1.0.x   | ✅        |

## 🚨 Reporting a Vulnerability

To report a vulnerability, [open a security issue on the project tracker](https://github.com/riverma/pds-agent-marketplace/issues/new).

For sensitive vulnerabilities that should not be disclosed publicly, please contact the maintainers privately via GitHub rather than filing a public issue.
